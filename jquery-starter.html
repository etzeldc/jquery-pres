$(selector).on(event, selector, function(eventObj), map) {

   //do something, or don't... .off() will negate it anyway.

});

  • "$": the "bling", starts it off showing its JQuery
  • "selector": identifies on which element the effect will take place
  • ".on()/.off()": the method used to either place or remove event handlers
  • "event": the action that will apply the effect to the selected element
  • "functon(eventObj)": specifies the function to run when the event occurs
  • "map": specifies an event map that may have more events to attach and functions to run
